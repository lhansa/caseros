# caseros

<!-- badges: start -->
<!-- badges: end -->

Una shiny app que cambiara para siempre el rumbo de los precios de
alquiler.

Los usuarios puntuan las casas en las que han vivido en el pasado. Asi
otros usuarios pueden aprovechar esa experiencia antes de entrar a vivir
en una casa nueva.
